# 🚚 RotaTracker API

SaaS de gestão logística e monitoramento de rotas.
Desenvolvido como projeto prático para consolidar arquitetura de software corporativa.

## 🚀 Tecnologias Utilizadas
* **Java 17 & Spring Boot 3**
* **Spring Security & JWT** (Autenticação Stateless e RBAC)
* **PostgreSQL & Flyway** (Migrations e Banco Relacional)
* **Docker** (Containerização)

## 🛡️ Segurança
A API conta com proteção de rotas por perfis de acesso (`ADMIN` e `MOTORISTA`), garantindo que apenas usuários autorizados possam modificar as viagens.
